$ come uso Claude Code

01
CLAUDE.md come contratto

Ogni progetto parte da un CLAUDE.md che definisce contesto, vincoli e convenzioni. Claude Code lo legge ad ogni sessione.

02
Analisi prima del codice

Chiedo a Claude Code di esplorare il codebase e proporre l'architettura prima di scrivere la prima riga.

03
Test generati insieme

I test non sono un afterthought: Claude Code li genera in parallelo all'implementazione.

04
Claude Desktop per l'intero workflow

MCP Filesystem, MCP Database e server personalizzati per connettere Claude Desktop a ogni sistema del progetto.
